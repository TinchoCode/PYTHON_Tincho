Español:
La diferencia entre los dos es que "V1" tiene solo 2 frames y la función de cambio funciona unicamente entre esos,

English:
The difference between the two is that "V1" has only 2 frames and the switch function works only between those, 
while in "V2" there are 3 frames and the switch function will work even if I add more frames to it but modifying some things.
Also specifying that the order in which the code was written is important to its function.
And that the use of "lambda:" is to create an anonymous function at the moment and be able to pass the argument that we want.

Now with V3 a change was made in the general structure of the program, making use of "class" for the interface itself and
leaves the creation of Frames and Buttons to functions, making the code less redundant.
