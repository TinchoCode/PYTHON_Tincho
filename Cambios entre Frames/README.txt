Español:
La diferencia entre los dos es que "V1" tiene solo 2 frames y la función de cambio funciona unicamente entre esos, mientras que en "V2" hay 3 frames y la funcion de cambio funcionaria incluso si le agrego mas frames.

English:
The difference between the two is that "V1" has only 2 frames and the switch function works only between those, while in "V2" there are 3 frames and the switch function will work even if I add more frames to it.
